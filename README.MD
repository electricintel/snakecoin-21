# quick start

1. 本地安装python 2.7 
2. pip安装flask
3. git clone git@github.com:shiyunlai/snakecoin.git 
4. cd snakecoin && python app.py

# test.http 
    
    snakecoin/test.http文件记录了snakecoin提供的服务.
    借助pychar的editor-base rest client工具,可直接进行测试
    其中包括三个方法
    1 创建一个交易
    2 开采新的区块
    3 查看区块
